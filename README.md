## Backend de App Delivery

Es el backend para desarrollar la API que nos permitirá todo lo que tiene que ver con la interacción de datos y cómo conectar con postgres y bases de datos


# Anotaciones

- En server.js está la IP, que en este caso es la misma del pc para pruebas, si no, con ipconfig en cmd la ipv4 es

- 'node server.js' con ese comando arranca el server

- Para detener la aplicación en la terminal, Ctrl + c
